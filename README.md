# JiaLiang-Zheng
A machine learning-based prediction model

## Code and Model Availability

To improve methodological transparency, we provide a subset of the code used in this study that implements the core data preprocessing and model training pipeline, with particular attention to preventing potential data leakage during imputation, feature selection, and cross‑validation. 

In addition, the final trained model identified as the best-performing algorithm in our study (Gradient Boosting) is released as an `.rds` file. This allows other researchers to directly load and reuse the trained model for validation, benchmarking, or methodological comparison in future studies.

These materials are shared to facilitate reproducibility and to support further research on predictive modeling of postoperative atrial fibrillation following CABG.
